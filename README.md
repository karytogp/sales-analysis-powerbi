# 📊 Sales Analysis Dashboard

## 📌 Descripción
Este proyecto consiste en un análisis de ventas desarrollado en Power BI.

## 🎯 Objetivo
Identificar tendencias de ventas, ciudades con mayor participación y categorías más rentables.

## 🛠 Herramientas utilizadas
- Power BI
- SQL
- Excel

## 📊 Análisis realizado
- Ventas por año
- Top 10 ciudades
- Categorías más vendidas
- Participación porcentual

## 📈 Resultados clave
- Technology lidera las ventas (~36%)
- Top 5 ciudades representan ~33%
- New York City es la ciudad principal
- Crecimiento sostenido desde 2016
